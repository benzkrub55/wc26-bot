# WC26 Bot — แจ้งเตือนบอลโลก 2026 ผ่าน Telegram

ระบบแจ้งเตือนเข้ากลุ่ม Telegram รันด้วย GitHub Actions (เซิร์ฟเวอร์ GitHub ไม่ต้องเปิดคอม)

## การแจ้งเตือน

| เวลา | อะไร | ไฟล์ |
|---|---|---|
| 00:01 น. | โปรแกรมแข่งของวันนี้ | `digest.py` |
| ~1 ชม.ก่อนเตะ | เตือนนัดที่กำลังจะเตะ + checklist (รวมนัดเวลาเดียวกัน) | `notify.py` |
| ~5 นาทีก่อนเตะ | ย้ำ checklist อีกครั้ง | `notify.py` |
| หลังจบเกม (ภายใน ~15 นาที) | สกอร์จบเกม รวมต่อเวลา/จุดโทษ | `results.py` |
| 23:59 น. | สรุปผลทุกนัดของวันนี้ | `digest.py` |

## ไฟล์

- `schedule.csv` — ตาราง 104 นัด เวลาไทย (อัพเดทชื่อทีมหลังจบแต่ละรอบ)
- `checklist.txt` — รายการงานทีม แก้ได้เลย บรรทัดละ 1 ข้อ
- `teams_th.py` — แมพชื่อทีมอังกฤษ→ไทย
- `sent_results.json` — ผลที่ประกาศไปแล้ว (workflow commit เอง ห้ามแก้มือ)
- `.github/workflows/` — ตัวตั้งเวลา 3 ตัว (notify / results / digest)

## Secrets (Settings → Secrets and variables → Actions)

| Name | ค่า |
|---|---|
| `BOT_TOKEN` | โทเคนบอทจาก @BotFather |
| `CHAT_ID` | chat id ของกลุ่ม (-100...) |
| `FOOTBALL_DATA_TOKEN` | API key จาก football-data.org (จำเป็นสำหรับผลบอล) |

## หมายเหตุ

- cron ของ GitHub อาจหน่วง 5–15 นาที: เตือน "1 ชม." มาช่วง ~50–65 นาทีก่อนเตะ, เตือน "5 นาที" เป็น best-effort
- ใช้ ~1,500–1,700 นาที Actions/เดือน — อยู่ในโควต้าฟรี private repo (2,000) แบบพอดี ๆ ถ้าจะเพิ่มความถี่อีกแนะนำเปลี่ยนรีโปเป็น public (ฟรีไม่จำกัด)
