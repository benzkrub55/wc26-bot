# WC26 Bot — แจ้งเตือนบอลโลก 2026 ผ่าน Telegram

แจ้งเตือนเข้ากลุ่ม Telegram ก่อนเตะทุกนัด (ภายใน 60 นาทีก่อนเริ่ม) พร้อมเช็คลิสต์ รันด้วย GitHub Actions ทุกชั่วโมงช่วง 22:00–11:00 เวลาไทย

## ไฟล์ในรีโป

- `schedule.csv` — ตารางแข่งทั้ง 104 นัด (เวลาไทย) อัพเดทชื่อทีมรอบน็อกเอาต์หลังจบแต่ละรอบ
- `notify.py` — สคริปต์เช็คตารางและส่งข้อความ Telegram
- `checklist.txt` — รายการเช็คลิสต์ที่แนบท้ายทุกข้อความ (แก้ไฟล์นี้ได้เลย บรรทัดละ 1 ข้อ)
- `.github/workflows/notify.yml` — ตัวตั้งเวลารัน

## การตั้งค่า (ทำครั้งเดียว)

1. ไปที่ Settings → Secrets and variables → Actions → New repository secret
2. เพิ่ม 2 ตัว:
   - `BOT_TOKEN` = โทเคนบอทจาก @BotFather
   - `CHAT_ID` = chat id ของกลุ่ม (ขึ้นต้นด้วย -100...)
3. ไปแท็บ Actions → กด enable workflow (ถ้ามีปุ่มถาม)

## ทดสอบ

แท็บ Actions → WC2026 Telegram Notifier → Run workflow — ถ้ามีนัดเตะใน 60 นาทีข้างหน้าจะมีข้อความเด้งเข้ากลุ่ม (ถ้าไม่มีนัดในช่วงนั้น log จะบอกว่า sent 0)

หมายเหตุ: cron ของ GitHub อาจหน่วงได้ราว 5–15 นาที ข้อความเลยมาถึงล่วงหน้าราว 5–55 นาทีก่อนเตะแล้วแต่นัด
