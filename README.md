# WC26 Bot — แจ้งเตือนบอลโลก 2026 ผ่าน Telegram

ระบบแจ้งเตือนเข้ากลุ่ม Telegram รันด้วย GitHub Actions (เซิร์ฟเวอร์ GitHub ไม่ต้องเปิดคอม)

## การแจ้งเตือน

1. **สรุปประจำวัน** (`digest.yml` → `digest.py`) — วันละ 4 รอบ เวลาไทย 00:01, 08:00, 16:00, 23:59: ผลนัดที่แข่งไปแล้ววันนี้ (สกอร์สดจาก football-data.org) + โปรแกรมที่เหลือของวัน และรอบ 00:01 แถมเตือน checklist งานประจำวัน
2. **ก่อนเตะทุกนัด** (`notify.yml` → `notify.py`) — เตือน 2 จังหวะ: ล่วงหน้า ~1 ชั่วโมง และ 1 นาทีสุดท้ายก่อนเขี่ยบอล พร้อม checklist ทีมงานทุกครั้ง

## ไฟล์

- `schedule.csv` — ตาราง 104 นัด เวลาไทย (อัพเดทชื่อทีมหลังจบแต่ละรอบ)
- `checklist.txt` — รายการงานทีม แก้ได้เลย บรรทัดละ 1 ข้อ
- `notify.py` / `digest.py` — สคริปต์แจ้งเตือน
- `.github/workflows/` — ตัวตั้งเวลา

## Secrets ที่ต้องตั้ง (Settings → Secrets and variables → Actions)

| Name | ค่า |
|---|---|
| `BOT_TOKEN` | โทเคนบอทจาก @BotFather |
| `CHAT_ID` | chat id ของกลุ่ม (-100...) |
| `FOOTBALL_DATA_TOKEN` | API key ฟรีจาก football-data.org (ใช้ดึงผลบอล ถ้าไม่ใส่ digest จะไม่มีสกอร์) |

## หมายเหตุ

- cron ของ GitHub อาจหน่วง 5–15 นาที ตัวเตือน "1 ชม.ก่อนเตะ" จึงมาช่วง ~35–65 นาทีก่อนเตะ ส่วน "1 นาทีสุดท้าย" เป็น best-effort (สคริปต์ตื่นก่อนแล้วนอนรอจนถึงเวลาจริง)
- ใช้ ~600–700 นาที Actions/เดือน อยู่ในโควต้าฟรีของ private repo (2,000 นาที/เดือน)
